# Media Cleaner Lite

A lightweight and high-performance WordPress plugin that safely detects unused media files, analyzes references across your website, and helps reclaim storage without risking accidental deletion.

## Features

- Detect unused media files
- Reference analysis
- WooCommerce support
- Elementor, Bricks, Beaver Builder & Divi support
- ACF & custom fields scanning
- Safe delete with verification
- Batch scanning
- REST API
- AJAX powered
- Translation ready
- RTL support
- Vanilla JavaScript

## Requirements

- WordPress 6.8+
- PHP 8.0+
